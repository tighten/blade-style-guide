# Blade Style Guide

> 1 space after start control structure directives
- `@if`
- `@elseif`
- `@for`
- `@foreach`
- `@endforeach`
- `@unless`
- `@forelse`


> No spaces after other directives
- `@endif`
- `@endfor`
- `@else`
- `@endforelse`
- `@endunless`
- `@section`
- `@endsection`
- `@show`
- `@yield`
- `@extends`
- `@parent`
- `@verbatim`
- `@empty`
- `@continue`
- `@break`
- `@php`
- `@endphp`
- `@include`
- `@includeIf`
- `@each`
- `@push`
- `@endpush`
- `@stack`

> 1 space on surrounding render directive content
- `{{ $x }}`
- `{!! $x !!}`
